# Java Tetris Game

Proyecto personal desarrollado durante la asignatura **Programación II** del Grado en Ingeniería Informática.

## Descripción
Implementación del clásico juego **Tetris** en Java.  
El jugador debe encajar piezas que caen, completar líneas y conseguir la mayor puntuación posible.  
Este proyecto permite practicar programación orientada a objetos, lógica de matrices y control de juego.

## Funcionalidades
- Movimiento y rotación de piezas
- Comprobación y eliminación de líneas completas
- Control de niveles y temporizador
- Interfaz gráfica desarrollada con Java Swing

## Tecnologías
- Java
- Swing
- Programación orientada a objetos (POO)
